# PdevGiveall
A Script to easily give items to the whole server

QBCORE AND QBOX ONLY

Drag into Standalone Folder

Enusre in CFG by typing  ensure PdevGiveall

Start the script and enjoy

--------------------------------------------------------------

/giveall type, itemname, ammount

List of types bellow

item
weapon
money

--------------------------------------------------------------

example bellow

/giveall item case_common 1

/giveall weapon weapon_appistol 1

/giveall money cash 1000
